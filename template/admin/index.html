git

  • $ git config --global user.name "John Doe"
  • $ git config --global user.email johndoe@example.com
  • $ git config --global core.autocrlf input
  • $ git config core.filemode false ←permission を無視する
  • $ git config credential.helper store ←認証情報の保存
  • $ git clone git@github.com::nttresonant/gooblog.git
  • $ git status
  • $ git add
  • $ git commit
  • $ git commit --amend
  • $ git reset HEAD
  • $ git cherry-pick 546de2cc727fcae02646c6af637f401e4238feef
  • $ git clean -n 削除されるであろうファイルを一覧表示する
  • $ git clean -f configに削除できない設定があっても、強制的に未追跡ファイルを削除する
  • $ git branch --set-upstream-to=origin/develop develop
  • $ git tag タグの一覧
  • $ git tag -a v1.0 -m 'message' タグの生成
  • $ git tag -a v1.0 -m 'message' (check sum) タグの後付
  • $ git tag -d v1.0 タグの削除
  • $ git log --abbrev-commit --pretty=oneline 省略形でログの一覧を表示する
