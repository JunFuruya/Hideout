# Project Title

Hideout

# Overview

Hideout is a web tool to help ME:
Hideout は自分用のWEBツールで、以下の用途で使用します。

* to accumulate my phisical and digital notes into a web system - 物理、電子データでのメモをWEBシステムに集約します。
* to organize my routines and financial activities - 業務やお金に関する活動をやりやすくします。

# Description

## 

* word list 単語帳
* link list リンク集
* slackbot  Slackからの命令で作業

## under construnction 製造中

* 取引先管理
* 見積書、請求書PDF作成
* 案件管理
